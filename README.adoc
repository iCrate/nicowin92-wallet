## Install

```
npm install
```

# Build

## Rebuild automatically
```
npm run build:watch
```

Open `build/index.html` with your browser

## Build once
```
npm run build
```

Open `build/index.html` with your browser


## Rebuild and start server
```
npm run start
```

Open `http://localhost:8080` with your browser.

## Run node

Wallet expects Web3 RPC at http://localhost:8545
